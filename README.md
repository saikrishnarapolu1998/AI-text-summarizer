# AI Text Summarizer

A simple Streamlit app that summarizes text using Hugging Face Transformers and the `facebook/bart-large-cnn` model.

## Features

- Enter text in a web interface
- Generate an AI-based summary
- Built with Streamlit, Transformers, and PyTorch

## Requirements

- Python 3.10 or 3.11 recommended

## Installation

```bash
git clone <your-repo-url>
cd ai-text-summarizer
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
